# Ghamau Nepal

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1xMJWnAVE6HFJVX_VTnkaoGIRiBinGViA

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app: `npm run dev`

## Deploy (GitHub Pages)

The Vite `base` path is set for this repository (`/shankar-daiiii/`). After `npm run build`, deploy the `dist` folder (for example `npm run deploy` if `gh-pages` is configured).
